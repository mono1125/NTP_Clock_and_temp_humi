The board can be fabricated in [pcbway](https://www.pcbway.jp)

## board info

- size: 98 x 75mm
- layer: 2
- Tickness: 1.6mm
- Material: FR-4 (TG150)
- Min Track: 6/6mil
- Min Hole Size: 0.3mm
- Edge connector: No
- Finished Copper: 1 oz Cu

Please feel free to use any of the following
- Solder Mask Color
